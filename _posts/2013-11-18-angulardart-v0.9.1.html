First AngularDart maintenance release v0.9.1 is now available!

AngularDart v0.9.1 elemental-mimicry includes a bug fix that removes unnecessary watchers which, in some cases (very large complex templates with large number of directives/components), gives a noticeable performance boost.

The release introduces an ability to define attribute mappings for components using annotations, which makes code much more readable. For examples see here.

In this release we also refactored the routing API and made it much easier to use: ng-bind-route is gone! RouteProvider instead will provide the route currently displayed by ng-view. We also made it easier to extract route parameters by adding parameters map to RouteProvider.

For full details of this release see the changelog.

Thanks to Anders Forsell for contributing PRs.
