--- title: 'Latest Dart community news' layout: post tags: - weekly category: dartwatch published: '2012-11-04T19:34:00+00:00' --- The latest news from around the Dart community (thanks to Seth Ladd for sub'ing while I've been on hols)

  • Dart Up and Running and Dart In Action are going to press!
  • Dart will be at Code Mash, Jan 8-13 in Sandusky Ohio. Indoor water park!!!
  • And we'll be at DevOxx in November and Rich Web Experience.
  • New presentations and video page. Slides, videos, more!
  • Editor gets more Code Cleanups, tracking more M1 and M2 features. abstract methods is one.
  • Videos from GOTO, including chats with Lars Bak and Anders Hejlsberg, and Giliad Bracha and Erik Meijer
  • Victor Savkin continues with Dart Casts. dartcasts.com. 3 episodes already up.
  • Speaking of Victor, he's also working on a DataBinder library. "implementing data binding via object mirrors". Speaking of mirrors... only Vm for now.
  • Lithuanian Dartisans are active with building their own translated site.
  • John Evans confirms that Dart Editor, pub, etc seem to be working on Windows 8.
  • Bernhard Pichler plays with Web Audio API and Dart, with his Dartflash library. Play Hey Jude in the browser!
  • Blossom, a startup working on Lean Product Management, has blogged about using Dart in their stack. Cool to see Dart used in production!
  • Chris Strom is back, though he'll say he never left. :) He's blogging about Dart again as he updates HipsterMVC for M1.
  • Kevin Moore releases his first early draft for a build system for Dart. Kevin is looking for feedback on hop.
  • Many of these libraries can be found in Pub. Let's check that site out!
