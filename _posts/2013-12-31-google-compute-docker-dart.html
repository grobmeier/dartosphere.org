--- title: 'Google Compute + Docker + Dart' layout: post tags: - dart - dartlang category: kevinwork published: '2013-12-31T10:52:00-08:00' --- Follow the instructions here: http://docs.docker.io/en/master/installation/google/

Assuming the last instruction works:

sudo docker run busybox echo 'docker on GCE \o/'

Run this:

sudo docker run sethladd/docker-dart dart --version

You should see something like:
