Episode 2: Library Imports

Dart has powerful mechanism for working with libraries. This screencast demonstrates how to declare, import, and re-export libraries.

Cheat Sheet:

import 'mylib.dart'
import 'mylib.dart' as mylib
import 'mylib.dart' show MyClass
import 'mylib.dart' hide MyClass
import 'mylib.dart' as mylib show MyClass
import 'mylib.dart' as mylib hide MyClass
export 'mylib.dart' show MyClass
export 'mylib.dart' hide MyClass
