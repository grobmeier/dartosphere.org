AngularDart release v0.9.3 reverse-telekinesis is now available!

This release includes basic support for browser cookies in a new service, allows specifying multiple css files in NgComponent, enables ng-model support for input[type=password] elements, allows ng-repeat over Iterables, improves digest performance and optimizes method calls on objects, supports a child scope parameter in the mock for compile, fixes a focus issue with input field handling, relieves GC pressure from watchCollection, correctly logs fired watches and supports inline templates in the expression extractor.

As always, for full details of this release, please see the changelog.
