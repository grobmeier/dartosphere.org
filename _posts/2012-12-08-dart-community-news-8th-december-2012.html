Thanks for the reminder +Seth Ladd that I'd not written a Dartwatch news update for some time - here's the latest Dartwatch update with news from the community. Exciting times as Milestone 2 issues are being resolved, and the libraries are being refactored. There are many new articles on the Dartlang website, and the Github 2012 Game competition has spawned some nice Dart games. There's something for everyone in this edition.

Dartlang News

  • Dartlang now has a Google+ "Dartisans" Community - Now we have another place to show and tell - come along and join in. It's a great place to announce your pub package, or other things you've been building with Dart.
  • +Adam Singer has also created a San Francisco Dart Community on Google+ - if you're in the area, join up.
  • Dart Web Components has been renamed "Web UI" to cover it's now expanded remit. Check out the latest awesome Dartians episode on this very subject:
    • Also, don't forget to check out the new Web UI Github Repo
  • In case you missed it, +Bob Nystrom's Dart language meeting notes announced that mixins are in serious discussion to adding to the Dart language spec. (If you're not sure what a mixin is, check out this stack overflow page "What is a mixin and why are they useful").
  • The Dart Editor now supports themes. The Dart Editor has a whole bunch of great features, but being able to theme it is great - especially when you spend large amounts of time with it. There's becoming fewer reasons to use another code editor.
  • SSL / TLS is now available in Dart, and +Adam Singer has created a Goo.gl URL shortener that uses SSL and SecureSocket
    • Check out the github repo
  • +Kevin Moore has updated his Pop Pop Win! dart game. The flying dart animations are great!
  • Drone.io is becoming a popular resource for continuous integration of Dart projects. Take a look and add a "Passing drone.io" badge to your github repo.
  • +Chris Strom has been continuing his epic series of blog posts about Dart - kudos for this!
  • ++Alexandre Ardhuin spotted that you can add <!-- language: lang-dart --> above your code snippets on Stack Overflow to get Dart syntax highlighting.
    • Check out the Dart tag on Stack Overflow
  • +John McCutchan has created a simple audio library for to help out games creators using Dart
    • Github repo
    • Don't forget to check out John's Dart Game Developers blog
  • If you're interested in isolates and async IO, then make sure you check out the new Streams experimental builds, described in depth here - it's currently living on the bleeding edge of the bleeding edge, but the Dart Team would love your feedback.

New Articles

  • Dart spec guru Gilad Bracha has posted a Dartlang article about Dart Mirrors. If you're interested in reflection, be sure to check it out.
  • +Kathy Walrath has posted a new Dartlang style guide for dartdoc comments. Please take the time to read it - consistency is good for everyone.
  • +Shailen Tuli has written a blog post about running unit tests in Dart. Make sure you also check out his article on the dartlang website about testing your pub packages

Github 2012 Game Compeition

  • +Dzenan Ridjanovic has been creating his Collision Clones game in Dart.

And Finally

While I've not been writing the Dartlang weekly newsletter, I've been putting together a Dart showcase website www.BuiltWithDart.com which shows off what we can build with Dart. If you have built an app, game, framework, or other infrastructure that uses or supports Dart, let me know, and provide a paragraph or so of text about your project, and I'll add it the site.
