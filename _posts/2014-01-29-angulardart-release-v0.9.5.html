AngularDart release v0.9.5 badger-magic is now available!

This release includes an expanded feature set for forms and ng-model, a much smaller dart2js output size and support for Dart SDK 1.2. As always, for full details of this release, please see the changelog.

Thanks to everyone who contributed PRs for this release:  Andrew Ryan Lazarus, Egor Day, Giovanni Silva, James deBoer, Kasper Lund, Matias Niemelä, Miško Hevery, PatrickJS, Pavel Jbanov, Pete Blois, Ted Sander, Victor Berchet.
